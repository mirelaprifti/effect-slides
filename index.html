How to use

01

Pick a direction

Open either template above. Both contain the same 14 slide types — only the visual system differs.

02

Set your details

Toggle Tweaks in the toolbar to set your talk title, name, handle, date and location once — it propagates to every slide.

03

Edit slide content

Slide markup is static HTML — click any heading, list item, or code line and type. Reorder via the thumbnail rail.

04

Drop in images

Speaker portrait and full-bleed image slots are drag-and-drop. Export to PPTX or PDF when you're ready.
